Skill 1 Writing Task 1 · identify the visual, then write fast

Real exam success starts with correctly identifying what you're looking at within seconds. Read each prompt and name the visual type from memory before checking.

Quick-fire identification

1"The chart shows population change in a city over a 50-year period."
2"The chart shows the proportion of a company's budget spent on five departments."
3"The diagram shows the stages by which paper is manufactured from wood."
4"The chart gives exact enrolment figures for four courses across three universities."
5"The maps show a town centre before and after a new shopping centre was built."
6"The chart compares the number of cars sold by four manufacturers in one year."
#Visual typeWhy
1Line graphChange over a continuous period — a line graph's core use.
2Pie chart"Proportion of a budget" = parts of a whole.
3Process diagram"Stages by which X is manufactured" = a sequence.
4Table"Exact figures" across two dimensions (course × university) suits a table's precision.
5Map comparison"Before and after" + physical location = a map.
6Bar chartComparing discrete categories (manufacturers) at one point in time.

Speed-overview drill · 10 min

Pick two of the six prompts above and write only the overview sentence for each — 2 sentences maximum, no data, 3 minutes per prompt. This tests whether the overview skeleton is now automatic.
